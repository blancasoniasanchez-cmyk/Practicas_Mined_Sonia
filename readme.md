## 🎯 ¿De qué trata este repositorio?

Este espacio está dedicado al **aprendizaje colaborativo** entre los docentes participantes del Bootcamp de Tecnología impulsado por el **MINEDUCYT**. Aquí encontrarás todas las prácticas, ejercicios, proyectos y recursos trabajados durante las sesiones del bootcamp.

> 💡 *"El conocimiento compartido es conocimiento multiplicado."*

## 📋 Cómo usar este repositorio

### 1️⃣ Clonar el repositorio
```bash
git clone https://github.com/tu-usuario/bootcamp-mineducyt.git
```

### 2️⃣ Navegar a una práctica
```bash
cd bootcamp-mineducyt/html/practica-01-estructura-basica
```

### 3️⃣ Abrir en tu editor favorito
```bash
code .   # VS Code
```

## 🤝 ¿Cómo contribuir?

¡Este repositorio es colaborativo! Si eres docente participante del bootcamp y quieres agregar tus prácticas o mejorar las existentes:

1. 🍴 Haz un **fork** del repositorio
2. 🌿 Crea una rama nueva: `git checkout -b mi-practica`
3. 💾 Guarda tus cambios: `git commit -m "✨ Añadir práctica de CSS Grid"`
4. 📤 Sube tu rama: `git push origin mi-practica`
5. 🔁 Abre un **Pull Request**
