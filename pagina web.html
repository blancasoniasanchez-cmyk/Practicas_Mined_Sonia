¿Qué es JavaScript?

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Es fundamental para crear interactividad en los sitios web.

Variables y Tipos

En el estándar moderno (ES6+), utilizamos let y const para declarar variables, dejando atrás el antiguo var.
